# video-player
Watch video with subtitles

This is a simple video player application that allows you to watch videos with subtitles. It supports various video formats and subtitle files.

## Features
- Play videos in different formats (MP4, AVI, MKV, etc.)
- Load and display subtitles (SRT, VTT, etc.)
- Basic playback controls (play, pause, stop, seek)
- Fullscreen mode
- Volume control
